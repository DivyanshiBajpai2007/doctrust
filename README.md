\# DocuTrust — Enterprise RAG Platform



Ask questions from any PDF document using AI.

Built with Python, LangChain, ChromaDB, Groq API and Streamlit.



\## Features

\- PDF upload and text extraction

\- Vector storage with ChromaDB

\- CRAG pattern for relevance grading

\- Query rewriting if retrieval fails

\- Source citations with every answer



\## Tech Stack

\- Python, Streamlit, LangChain

\- ChromaDB, FastEmbed

\- Groq API (Llama 3.3 70B)
